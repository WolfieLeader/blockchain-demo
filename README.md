# Blockchain Demo in Node.js

A simple blockchain and cryptocurrency wallet implemented in Node.js and TypeScript (for learning purposes).

Watch the [Blockchain in 100 Seconds](https://youtu.be/qF7dkrce-mQ) video.

# Usage

```
git clone <this-repo>

npm install
npm start
```
